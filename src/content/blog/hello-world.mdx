Welcome to my blog! This is the first post on my new personal site.

## Why I Started This Blog

I've been meaning to write more publicly for a while. This blog is a place where I'll share my thoughts on technology, engineering, research, and whatever else I find interesting.

## What to Expect

I plan to write about:

- **Technical deep-dives** — detailed explorations of interesting problems
- **Project updates** — progress on things I'm building
- **Industry thoughts** — observations and opinions on tech trends
- **Research** — summaries and discussions of interesting papers

## How This Site Works

This site is built with [Astro](https://astro.build) and deployed to AWS (S3 + CloudFront). Blog posts are written in MDX — Markdown with support for embedded components — and committed directly to the git repo. When I push to main, GitHub Actions builds and deploys automatically.

The entire site is static, fast, and cheap to host. No databases, no server-side rendering, no JavaScript unless absolutely necessary.

> The best tool is the one you actually use. I wanted something simple enough that writing a post is just writing a file.

Stay tuned for more posts!
